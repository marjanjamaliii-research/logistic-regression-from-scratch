# logistic-regression-from-scratch


This project implements a simple logistic regression model using only NumPy.

Overview

The goal of this project is to understand how a basic machine learning algorithm works without relying on high-level libraries such as PyTorch or TensorFlow.

The model predicts binary outcomes (0 or 1) based on input features.

Key Concepts

- Sigmoid function to convert outputs into probabilities
- Binary cross-entropy loss
- Gradient descent for optimization

Implementation Details

- The model is implemented from scratch using NumPy
- A synthetic dataset is generated for training
- The training loop updates weights iteratively to minimize loss

Results

The model successfully learns to classify the data and achieves a reasonable accuracy.

Technologies Used

- Python
- NumPy
- Matplotlib

How to Run

1. Open the notebook
2. Run all cells
3. Observe the loss curve and accuracy
